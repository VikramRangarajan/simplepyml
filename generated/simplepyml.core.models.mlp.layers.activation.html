simplepyml.core.models.mlp.layers.activation#

class simplepyml.core.models.mlp.layers.activation.Activation(activation: Callable[[ndarray], ndarray], *args, **kwargs)[source]#

Bases: Layer

Activation Layer.

Definition:

Takes an input, applies an activation function to it.

\[Y = \phi(X)\]

To calculate the loss gradient w.r.t. the input, we use this formula:

\[\frac{\partial L}{\partial X} = \frac{\partial L}{\partial Y} \odot \phi'(z)\]
Parameters:

Activation function. See activation functions
